# Investigations

This directory now holds only active or recent investigation files.

## Usage

- Confirmatory, model-scoped validation belongs in probes at `.kb/models/{name}/probes/`.
- Novel exploration with unknowns still belongs in investigation files here.

## Archival

- Older investigations are moved to `.kb/investigations/archived/` as read-only provenance.
- Model citations should point to archived files when an investigation has been moved.
