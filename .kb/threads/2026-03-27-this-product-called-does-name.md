# Intent arrives without shape — how does the system help you find the right first move?

## 2026-03-27

Dylan noticed a recurring pattern: he arrives with clear intent ('name this product', 'fix this thing', 'understand this') but doesn't know the topology — is this a search problem (explore), a convergence problem (plan), a forming question (thread), or ready to execute (issue)? Sometimes he can sense it, sometimes he's surprised. The orchestrator should be classifying this, but the deeper question is: is this a product-level concern? If the system helps you think, shouldn't it also help you figure out *how* to start thinking about something?

Connection to behavioral grammars: the model's core finding was that skills are 'a way of thinking' — a grammar that shapes action. Claim 5 says grammars can't detect their own failures; the agent experiences constraint as judgment. Dylan is describing the same phenomenon from the human side: the routing grammar (explore vs plan vs thread vs issue) is invisible. When it works, it feels like good instinct. When it fails, he's surprised but can't see why. The Feb 24 probe showed identity ('I know what I am') diverges from action ('I know what to do') — and the agent can't see the gap. Dylan's experience mirrors this: he knows the destination (identity/intent) but not the topology (action/shape). The question forming: if the product makes thinking legible, should it also make this routing grammar legible? Not just do the right thing behind the scenes, but help the human develop their own sense of work shape — the way the behavioral grammar model helped us see why skills fail?

The insight crystallized: the product's actual front door is the shape classification moment, not the cycle diagram. The method guide (must-ship from the release bundle investigation) should open with this: you arrive with intent, the system helps you see its shape. The shape taxonomy is small — search (multiple unknowns), convergence (pieces exist, need sequencing), forming (question not sharp yet), execution (scope clear). Teaching the system IS making this moment legible. One sentence of shape classification in every response, not a framework — the user internalizes the grammar through exposure, not instruction. This is behavioral grammar Claim 7 (human-agent coupling) used intentionally: the coupling should develop the human's grammar, not just the system's routing accuracy.
