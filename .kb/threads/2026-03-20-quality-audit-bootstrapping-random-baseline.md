# Quality audit bootstrapping — random baseline produces the data that teaches the system what to audit

## 2026-03-20

Random audit (N%) → first negative signal data → correlations emerge (which skills/models/patterns fail) → audit selection shifts from random to risk-weighted. The measurement system evolves from data it produces, not upfront theory. Same pattern as the 4 interventions that work: embed the feedback loop structurally, don't try to predict what needs review.
