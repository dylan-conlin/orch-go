## Why This Template Was Deprecated

This template was designed to synthesize patterns across multiple investigations. In practice:

1. **Compounded errors** - 4 wrong investigations became 1 authoritative-looking wrong synthesis
2. **Created false confidence** - Sophisticated framework made wrong answers look rigorous
3. **Institutionalized wrong conclusions** - Investigation → Synthesis → Decision pipeline propagated errors

**Case study:** Codex hang synthesis concluded "frame problem" with sophisticated analysis. Actual cause: simple file size limit.

**What to do instead:** Use simple investigations. Test hypotheses. Don't synthesize.
