## orch-go

OpenCode orchestration CLI

### Synopsis

orch-go is a CLI tool for orchestrating OpenCode sessions.

It provides commands for spawning new sessions, sending messages to existing
sessions, and monitoring session events via SSE.

```
orch-go [flags]
```

### Options

```
  -h, --help            help for orch-go
      --server string   OpenCode server URL (default "http://localhost:4096")
```
