## orch-go servers stop

Stop servers for a project (legacy)

### Synopsis

Stop the servers tmux session for a project.

**Note:** This is a legacy command. Consider using `orch servers down` for launchd-based
server management.

This kills the `workers-{project}` tmux session.

```
orch servers stop <project> [flags]
```

### Examples

```bash
orch servers stop myproject
```
