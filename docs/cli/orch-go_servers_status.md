## orch-go servers status

Show server status

### Synopsis

Show server status.

Without a project argument, shows a summary of all projects.
With a project argument, shows per-server status from `servers.yaml`.

```
orch servers status [project] [flags]
```

### Examples

```bash
# Summary view (all projects)
orch servers status

# Per-server status for a project
orch servers status myproject
```

### Output (Summary)

```
Use 'orch servers list' for detailed view
```

**Status indicators:**
- `●` running - Server is actively running
- `○` stopped - Server is not running
- `✗` error - Server encountered an error
