### Function Details

**Function Signature:**
```go
func FormatTable(headers []string, rows [][]string) string
```

**Behavior:**
- Renders headers and rows as an aligned text table
- Auto-sizes each column based on widest content (header or data)
- Separates header from data rows with a line of dashes
- Correctly handles ANSI escape codes using existing StripANSI function
- Handles edge cases: empty rows, mismatched column counts, nil/empty input

**Design Choices Made:**
- Separator style: pipe (|) with single space padding on both sides (e.g., " | ")
- Header separator: dashes with dash-plus-dash (-+-) between columns
- Rows with fewer columns than headers: padded with empty cells
- Rows with more columns than headers: included with extra columns
