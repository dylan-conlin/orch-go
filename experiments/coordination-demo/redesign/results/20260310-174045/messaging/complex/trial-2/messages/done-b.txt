### Implementation Details

**File: pkg/display/display.go**
- Added FormatTable(headers []string, rows [][]string) string
  - Auto-sizes columns based on widest content
  - Uses StripANSI for correct ANSI width calculation
  - Handles rows with more columns than headers (includes extra columns)
  - Handles rows with fewer columns than headers (pads with empty cells)

- Added helper functions:
  - visualWidth(s string) int - Returns visual width ignoring ANSI codes
  - buildTableRow(cells []string, colWidths []int) string - Formats single row
  - buildSeparatorRow(colWidths []int) string - Creates dash separator line
  - padToWidth(s string, width int) string - Right-pads with spaces while preserving ANSI

### Design Choices Made

1. **Column Borders:** Used pipe separators (|) for clarity and alignment
2. **Padding:** Single space on each side of content (" | ")
3. **Separator:** Dashes (---) for header/data separation
4. **Extra Columns:** Rows with more columns than headers are included (not truncated)
5. **ANSI Handling:** ANSI codes preserved in output, but visual width calculated without them
