==============================================

Insertion Points:

- Tests: After TestFormatDurationShort (line 134)

New Functions to Add:

1. VisualWidth(s string) int
   - Returns visual display width of a string, ignoring ANSI escape codes
   - Uses existing StripANSI() function to remove ANSI codes
   - Counts runes (not bytes) for correct Unicode handling
   - Implementation: strip ANSI, convert to runes, return len(runes)

2. PadToWidth(s string, width int) string
   - Right-pads string with spaces to reach target visual width
   - ANSI codes preserved but don't count toward width
   - If string already wider than width, returns unchanged
   - Implementation: calculate visual width, pad with spaces as needed

Test Coverage:
- TestVisualWidth: plain ASCII, ANSI colors, Unicode (emoji, CJK), empty string, edge cases
- TestPadToWidth: plain strings, ANSI colored, Unicode, edge cases (already at width, wider than target)
