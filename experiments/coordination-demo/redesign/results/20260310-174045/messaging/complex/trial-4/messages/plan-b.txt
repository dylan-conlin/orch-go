Function signature:
  func FormatTable(headers []string, rows [][]string) string

Design choices:
  - Border style: Pipe (|) and dash (-) separators
  - Padding: 1 space on each side of content
  - Header separator: Full line of dashes
  - Column alignment: Left-aligned
  - ANSI handling: Use StripANSI for width calculation
  - Edge cases: Handle empty rows, mismatched column counts

Implementation approach:
  1. Calculate column widths from headers + all rows (use StripANSI)
  2. Format headers with padding and pipes
  3. Create separator line
  4. Format each row with padding and pipes
  5. Handle rows with fewer columns than headers (pad with empty strings)

  - Empty table (headers only)
